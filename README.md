## 🚀 البدء السريع

### المتطلبات
```bash
Node.js 18+
npm أو yarn
```

### التثبيت
```bash
# استنساخ المشروع
git clone https://github.com/ayhamdiiko-afk/OmegaShield.git

# الدخول للمجلد
cd OmegaShield

# تثبيت المكتبات
npm install
```

### التشغيل
```bash
# تشغيل في بيئة التطوير
npm run dev

# سيعمل على http://localhost:5173
```

## 📁 البنية

```
src/
├── components/
│   ├── Logo.jsx              # الشعار الملكي
│   ├── Header.jsx            # شريط الرأس
│   ├── RoomCard.jsx          # بطاقة الغرفة
│   └── FloatingGifts.jsx     # الهدايا المتحركة
├── pages/
│   ├── Login.jsx             # صفحة التسجيل
│   ├── Home.jsx              # الصفحة الرئيسية
│   ├── VoiceRoom.jsx         # غرفة الصوت
│   └── Profile.jsx           # ملف المستخدم الشخصي
├── App.jsx                   # التطبيق الرئيسي
├── main.jsx                  # نقطة الدخول
└── index.css                 # الأسلوب العام
```

## 🛠️ التقنيات المستخدمة

- **React.js** - مكتبة الواجهة
- **Tailwind CSS** - تصميم الواجهة
- **Framer Motion** - الحركات والتأثيرات
- **React Icons** - الرموز والأيقونات
- **Vite** - بناء المشروع
- **React Router** - التنقل بين الصفحات
